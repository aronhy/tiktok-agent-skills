# TikTok 增长计划报告模板

仅填入上游来源账本和增长差距矩阵支持的信息。报告必须结果前置：先定位、进入决策和动作，后写证据、路径和限制。将观察、推断、假设和待验证项分开；缺失字段保留“未提供”“未公开”或“读取失败”，不按零补值、不用行业基准补造。

本模板只在四项必需输入齐全，或用户在首次反问后明确表示不能/不愿补充并要求继续时使用。后一情况是**有界部分输出**：所有受缺失输入影响的标题仍保留，但结论填“暂无法判断”，开头写明缺失输入、影响、假设和未运行的上游研究，整份报告标为 C（低可信度）；只填写不依赖缺失项的条件框架和可逆动作。

必需输入不足且用户显式要求继续：**暂无法判断。** 缺失：`{}`；影响：`{}`；假设：`{}`；未运行的上游研究：`{}`。（可信度：C，低）

定位依据：`{账号资产 + 类目机会 + 商业目标}`。反证/关键前提：`{}`。

## 10. 可直接复制给 AI 的下一步任务

```text
已知事实与来源：{仅填来源账本支持的数据、市场、窗口和可信度}。
目标：{下一个最重要的验证/执行目标}。
限制：{预算、团队、库存、合规、数据或访问限制}。
请输出：{明确交付物，例如商品验证表、视频脚本实验、达人类型筛选或复盘表}。
规则：只使用已提供或可公开验证的信息；将观察、推断、假设和待验证项分开；缺失数据写“未提供”“未公开”或“读取失败”，不得编造。
```

可按需要提供 1–3 个下一步任务，优先补齐会改变定位、进入决策或首要行动的关键缺口。

- **A：** 账号和类目关键结论均有充分、可追溯且口径兼容的跨来源证据。
